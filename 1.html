HTML

HyperText Markup Language (HTML) 는 웹 브라우저에 표시되도록 설계된 문서의 표준 마크업 언어이다. CSS(Cascading Style Sheet)와 같은 기술과 자바스크립트와 같은 스크립트 언어의 도움을 받을 수 있다. 웹 브라우저는 웹 서버 또는 로컬 스토리지로부터 HTML 문서를 수신하여 멀티미디어 웹 페이지로 문서를 렌더링합니다. HTML은 웹 페이지의 구조를 의미론적으로 그리고 원래 문서의 모양에 대한 단서를 포함합니다. HTML 요소는 HTML 페이지의 구성 요소입니다. HTML 구성을 사용하면 이미지 및 대화형 양식과 같은 기타 객체를 렌더링된 페이지에 포함할 수 있습니다. HTML은 제목, 단락, 목록, 링크, 인용문 및 기타 항목과 같은 텍스트에 대한 구조적 의미를 표시함으로써 구조화된 문서를 만드는 수단을 제공한다. HTML 요소는 꺾쇠 괄호를 사용하여 작성되는 태그로 설명됩니다. ,와 같은 태그는 페이지에 콘텐츠를 직접 소개한다. [p]와 같은 다른 태그는 문서 텍스트에 대한 정보를 서라운드하고 제공하며 하위 요소로 다른 태그를 포함할 수 있습니다. 브라우저는 HTML 태그를 표시하지 않고 페이지의 내용을 해석하는 데 사용합니다. HTML은 자바스크립트와 같은 스크립팅 언어로 작성된 프로그램을 내장할 수 있으며 웹 페이지의 동작과 내용에 영향을 미친다. CSS의 포함은 콘텐츠의 모양과 레이아웃을 정의합니다. W3C(World Wide Web Consortium)는 HTML의 전 유지자이자 CSS 표준의 현재 유지자로서 1997년부터 명시적 표현 HTML에 대한 CSS 사용을 권장하고 있다.[2] HTML5로 알려진 HTML의 한 형태는 자바스크립트와 협력하여 비디오와 오디오를 표시하는 데 사용된다.
