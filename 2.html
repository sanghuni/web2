CSS

CSS(Cascading Style Sheets)는 HTML과 같은 마크업 언어로 작성된 문서를 기술하는 데 사용되는 스타일시트 언어이다.[2] CSS는 레이아웃, 색상, 글꼴을 포함한 프리젠테이션과 콘텐츠를 분리할 수 있도록 설계되었다.[3] 이렇게 분리하면 컨텐츠 접근성이 향상되고 프리젠테이션 특성에 대한 유연성과 제어력이 향상되며, 별도의 .css 파일에 관련 CSS를 지정하여 형식을 여러 웹 페이지가 공유할 수 있으므로 구조 컨텐츠의 복잡성과 반복이 줄어듭니다. ed는 파일을 공유하는 페이지와 파일 형식 간의 페이지 로드 속도를 향상시킵니다. 포맷과 콘텐츠의 분리는 또한 화면, 인쇄, 음성 기반(음성 기반 브라우저 또는 화면 리더를 통한) 및 점자 기반 촉각 장치와 같은 다른 렌더링 방법에 대해 동일한 마크업 페이지를 다른 스타일로 표시하는 것을 가능하게 한다. CSS는 또한 모바일 장치에서 컨텐츠에 접근할 경우 대체 포맷을 위한 규칙을 가지고 있다.[4] Cascading이라는 이름은 둘 이상의 규칙이 특정 요소와 일치하는 경우 적용할 스타일 규칙을 결정하기 위해 지정된 우선순위 체계에서 유래되었습니다. 이 계단식 우선순위 체계는 예측 가능하다. CSS 규격은 월드 와이드 웹 컨소시엄(W3C)에 의해 관리된다. 인터넷 미디어 타입 (MIME 타입) text/css는 RFC 2318(1998년 3월)에 의해 CSS와 함께 사용하도록 등록되어 있다. W3C는 CSS 문서에 대한 무료 CSS 검증 서비스를 운영하고 있다.[5]
